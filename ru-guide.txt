# Установка:
1. Скачать и установить Python с Microsoft Store (версия 3.10)
2. Открыть Command Port (Командную строку)
3. Ввести "pip install pyautogui" и нажать Enter


# Запуск:
1. В проводнике перейти в корневую папку приложения (вы должны увидеть файл bot.py)
2. Нажать вне файлов правой кнопкой мышки -> Открыть Command Port (Командную строку)
3. Запустить бота данной командой "python bot.py"


# Флаги (добавлять через пробел "python bot.py -флаг1 -флаг2"):
-party
(создает пати 5v5)

-radiant
(использовать, когда игра уже идет и игроки 1-5 играют на Свет)

-dire
(использовать, когда игра уже идет и игроки 1-5 играют на Тьму)
